# 487. Precioso É Jesus

[Verse 1]
Anelo por Cristo, meu Rei Salvador;
As honras a ele darei com louvor,
Porque sempre achei nele graça e favor;
Precioso é Jesus para mim!

[Verse 2]
(Estribilho)
Precioso é Jesus para mim!
Precioso é Jesus para mim!
Celeste prazer é Jesus conhecer!
Precioso é Jesus para mim!

[Verse 3]
Afável, à porta do meu coração
Humilde e mui manso aguardou admissão
Do triste pecado outorgou-me perdão;
Precioso é Jesus para mim!

[Verse 4]
Firmado em Jesus eu alcanço afinal
Futuro de glória na vida eternal,
Sem uma desdita, nem nuvem de mal;
Precioso é Jesus para mim!

[Verse 5]
Na glória, minha alma deseja gozar,
E mais com o Amado se identificar;
Com todos os salvos servir e adorar;
Precioso é Jesus para mim!
