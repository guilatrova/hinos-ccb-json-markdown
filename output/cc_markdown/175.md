# 175. Minha Aspiração

[Verse 1]
Cristo, bom Mestre, eis meu querer;
Tua vontade sempre fazer;
Faze-me forte pra resistir
Duras fraquezas que possam vir.

[Verse 2]
Cristo, bom Mestre, eis meu querer;
Mais santidade de vida ter;
Faze-me firme, Cristo, meu Deus,
Para não deixar a senda dos céus.

[Verse 3]
Cristo, bom Mestre, eis meu querer;
Todas as minhas faltas vencer;
Faze-me rijo para lutar,
Para a vitória sempre ganhar.
